# mvcc.extend

Copies properties from one object to another.

## Usage

mvcc.extend(target, source);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| target       | `Object`   | Target object.                     |
| source       | `Object`   | Source object.                     |

### Returns

`boolean` Reference to 'target'
