# mvcc.isBoolean

The `isBoolean` function determines if the reference is a `Boolean`.

## Usage

mvcc.isBoolean(value);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| value        | `*`        | The reference to check.            |

### Returns

`Boolean` The function returns true if the `value` is a `Boolean`.
