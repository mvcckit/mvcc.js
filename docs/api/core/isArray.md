# mvcc.isArray

The `isArray` function determines if the reference is an `Array`.

## Usage

mvcc.isArray(value);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| value        | `*`        | The reference to check.            |

### Returns

`Boolean` The function returns true if the `value` is an `Array`.
