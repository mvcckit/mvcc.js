# mvcc.isString

The `isString` function determines if the reference is a `String`.

## Usage

mvcc.isString(value);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| value        | `*`        | The reference to check.            |

### Returns

`Boolean` The function returns true if the `value` is a `String`.
