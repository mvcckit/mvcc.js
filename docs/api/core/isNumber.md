# mvcc.isNumber

The `isNumber` function determines if the reference is a `Number`.

## Usage

mvcc.isNumber(value);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| value        | `*`        | The reference to check.            |

### Returns

`Boolean` The function returns true if the `value` is a `Number`.
