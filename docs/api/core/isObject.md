# mvcc.isObject

The `isObject` function determines if the reference is an `Object`.

## Usage

mvcc.isObject(value);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| value        | `*`        | The reference to check.            |

### Returns

`Boolean` The function returns true if the `value` is an `Object`.
