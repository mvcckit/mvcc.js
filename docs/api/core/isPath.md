# mvcc.isPath

The `isPath` function determines if the reference is a route path.

## Usage

mvcc.isPath(value);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| value        | `*`        | The reference to check.            |

### Returns

`Boolean` The function returns true if the `value` is a route path.
