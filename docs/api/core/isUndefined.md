# mvcc.isUndefined

The `isUndefined` function determines if the reference is undefined.

## Usage

mvcc.isUndefined(value);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| value        | `*`        | The reference to check.            |

### Returns

`Boolean` The function returns true if the `value` is undefined.
