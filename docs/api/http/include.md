# mvcc.http.jsonp

Include external content in a container.

## Usage

mvcc.http.include(url, el).then(success, [failure]);

### Arguments

| Parameter    | Type       | Details                            |
| ------------ | ---------- | ---------------------------------- |
| url          | `string`   | The url of the page or file.       |
| el           | `Element`  | The container.                     |
